# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
through [GitHub Security Advisories](https://github.com/metantesan/api-gateway/security/advisories).

Please do **not** report security vulnerabilities through public GitHub issues,
discussions, or any other public channel.

## What to Include

When reporting, please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Affected versions, if known
- Any potential mitigations you have identified

## Disclosure Policy

We follow [coordinated disclosure](https://en.wikipedia.org/wiki/Responsible_disclosure)
and will work with you to resolve the issue before any public announcement is
made.

## Supported Versions

Only the latest release is actively supported. Please update to the newest
version before reporting vulnerabilities.
